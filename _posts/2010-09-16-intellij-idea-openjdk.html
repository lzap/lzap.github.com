Pokud by snad někdo chtěl provozovat vývojové prostředí IntelliJ IDEA na OpenJDK, tak mám malou radu. Nedělejte to. Program se chová nestandardně, některé funkce jsou zcela nedostupné, občas obdržíte oznámení o pádu (program však obvykle "drží") a vše je opravdu (ale *opravdu*) hodně pomalé. Ne nadarmo podpora společnosti JetBrains nabádá k používání oficiální Javy od Oracle. Uff. Nějak mi to ještě stále nejde "do huby" ;-)

A jak zjistit, že máte tu správnou Javu? Začal bych příkazem java -version v příkazové řádce. Pokud tam najdete něco jako OpenJDK, je to špatně. Pokud něco jako Java HotSpot(TM), je to správně. Podobně to můžete zjistit přímo v programu v nabídce Help - About.

Standardní součástí dnešních Linuxů bývá z licenčních důvodů právě OpenJDK. Jak si nainstalovat "originální" Javu najdete v dokumentaci ke svým distribucím. Stačí vygooglit "Sun Java Ubuntu" či "Sun Java Fedora" a kliknout na první odkaz.

You have been warned!

Update: Mám zprávy, že IDEA se dá na OpenJDK dlouhodobě provozovat. Jen je potřeba vynaložit vyšší úsilí, pohrát si s nastavením fontů a možná též i něco opatchovat ;-) Další info Lukáš Vlček.
