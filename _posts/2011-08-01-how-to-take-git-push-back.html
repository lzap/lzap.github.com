Ouch, just pushed something I don't want in my remote git repo. Easy help:

# git reset --hard HEAD~1
# git push origin +master

Oops, I also pushed a tag. Well, it's a git, yea?

# git tag -d MY_TAG
# git push origin :refs/tags/MY_TAG

Use with care.
