Potřebujete rozchodit oficiální Javu (od Sunu) v prohlížeči Google Chrome v novém Ubuntu (například kvůli přístupu do banky)? Je to snadné.

Nejprve nainstalujeme Javu od Sunu:

agi sun-java6-bin sun-java6-fonts sun-java6-jdk sun-java6-jre sun-java6-plugin

Nyní zvolíme tuto Javu jako výchozí:

sudo update-alternatives --config java

V nabídce zvolíme Javu od Sunu. Toto si můžeme ihned ověřit (není nutný žádný restart terminálu):

$ java -version
java version "1.6.0_20"
Java(TM) SE Runtime Environment (build 1.6.0_20-b02)
Java HotSpot(TM) Client VM (build 16.3-b01, mixed mode, sharing)

No a restartujte Google Chrome. Na stránce about:plugins můžete ověřit, zda Java funguje. Hotovo. Říkal jsem to, že to bude brnkačka.

ps - agi = alias apt-get install
