Disable Window Animations updated for Fedora 17

May 30, 2012

  • gnome-shell

Few weekends ago, I've updated my Gnome Shell plugin Disable Window Animations for Fedora 17 (Gnome Shell 3.4). If you switch workspaces often, you might find this handy. I switch them thousands time a day, this saves me time & eyes. Like it? Say thanks in the comments.

This extension simply disable all window manager animations: window "slide-ins" and workspace "slide-ups" and "slide-downs". However, it does not disable dash animations ("zoom-ins" and
"zoom-outs" etc.) because I don't find them annoying. For more info you can visit my github.com homepage.
