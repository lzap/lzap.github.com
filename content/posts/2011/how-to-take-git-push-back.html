--- type: "post" aliases: - /2011/08/how-to-take-git-push-back.html date: "2011-08-01T00:00:00Z" tags: - git title: How to take a git push back --- Ouch, just pushed something I don't want in my remote git repo. Easy help:

# git reset --hard HEAD~1
# git push origin +master

Oops, I also pushed a tag. Well, it's a git, yea?

# git tag -d MY_TAG
# git push origin :refs/tags/MY_TAG

Use with care.
