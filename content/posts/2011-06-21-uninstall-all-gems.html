--- type: "post" aliases: - /2011/06/uninstall-all-gems.html date: "2011-06-21T00:00:00Z" tags: - ruby - gem title: Uninstall all the gems --- From time to time I need to uninstall all my gems within RVM gemset. I googled a nice command for this:

# gem list | cut -d" " -f1 | xargs gem uninstall -aIx

Then I usually go for:

# gem install bundler && bundle install

Easy peasy. Kudos to geekystuff.net. Thanks.
